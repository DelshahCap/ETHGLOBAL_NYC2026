# specs/

This folder holds the prompts, specifications, and planning artifacts that
directed the AI on this project — the human-authored intent behind the code.

Keep here:
- **Prompts** — notable instructions given to Claude Code / other AI tools.
- **Specs** — feature and contract specifications (escrow vault, yield position,
  release logic, oracle interface, NYC Open Data / HPD integration).
- **Planning** — design notes, decisions, and task breakdowns.

See [../AI_USAGE.md](../AI_USAGE.md) for the running log of where AI assisted and
which files each session touched, and [../CLAUDE.md](../CLAUDE.md) for project
context.
